## How to Run the code

All the codes for this assignment is written in `a2.ipynb` file. You can run the following notebook from top to bottom. 

If error occurs please check if all the dependent libraries are installed in the kernel which is running the notebook.

The required libraries includes:

```
numpy
matplotlib.pyplot
torch
torchvision
```

you could install them by typing following command in the terminal.

```
pip install numpy
pip install matplotlib
```

and download pytorch, and torchvision library from https://pytorch.org/get-started/locally/
